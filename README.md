Очистка датасета с пропущенными значениями
Работал с синтетическим датасетом розничных товаров, где пропусков было — хоть отбавляй. В некоторых колонках до 63% NaN. Задача была привести это в порядок и не потерять полезную информацию там, где это возможно.
Выводы:
Больше половины товаров без категории — грустно, но хотя бы теперь это явно видно.
Категория B заметно дороже остальных. Если это не артефакт данных, то там явно премиум-сегмент.
Рейтинг скидочных товаров оценивают чуть выше.
